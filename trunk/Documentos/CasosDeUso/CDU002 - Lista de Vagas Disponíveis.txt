************************************************************************
CDU002 - Lista de Vagas Disponíveis
************************************************************************

Atores
========================================================================
usuário não cadastrado, usuário cadastrado LogJobs FREE, usuário cadastrado LogJobs VIP


Pré-condições
========================================================================
Acesso ao site LogJobs.
Existência de dados de acordo com o critério de pesquisa informado.


Pós-condições
========================================================================
Apresentar uma lista pagina de vagas ativas no LogJobs de acordo com o critério de pesquisa informado.


Variações Tecnológicas
========================================================================
A lista de pesquisa poderá ser visualizada via internet, aplicativo desktop, gadget de sistema operacional e site de internet mobile.


Questões em aberto
========================================================================
O sistema de paginação deverá ser trocado por um modelo incremental de apresentação dos dados?
Caso a pesquisa não encontre resultados de acordo com o critério, poderá sugerir resultados aproximados?


Fluxo Principal
========================================================================

1. O usuário visualiza a lista de vagas disponíveis de acordo com o critério de pesquisa informado no "CDU001 - Pesquisar Vagas"

	1.1. O usuário poderá refinar o resultado da pesquisa de acordo com o "CDU003 - Refinar Pesquisa"
	
	1.2. Caso o usuário esteja autenticado como PREMIUM e já tenha visto a vaga, será apresentado um icone destacando que aquela vaga já foi visualizada ou não	
	
	1.3. Caso o usuário esteja autenticado como PREMIUM e já esteja concorrendo à vaga, será apresentado um ícone indicando que o currículo já foi enviado "X veze(s)"

2. O site irá apresentar as seguintes informações: Título do Anúncio, Descrição resumida, Quantidade de Vagas, Código, Data, Cidade, Estado, Faixa Salarial e um link "Mais Informações".

	2.1. Caso a vaga seja do tipo PREMIUM, ela será apresentada nas primeiras posições e em destaque (borda e cor de fundo diferenciado).

		2.1.1. Caso haja mais de uma vaga PREMIUM para o critério informado, o desempate de posicionamento será pela data da vaga.

	2.2. Caso o usuário esteja autenticado como FREE, será apresentado um botão "Concorrer a esta vaga" caso a vaga seja FREE ou um botão "Seja usuário premium e concorra a esta vaga", caso a vaga seja PREMIUM

	2.3. Caso o usuário esteja autenticado como PREMIUM, será apresentado um botão "Concorrer a esta vaga".

3. Ao clicar no título do anúncio ou no link "Mais Informações", o usuário será redirecionado para a página de detalhes do anúncio.

4. A página de detalhes do anúncio irá apresentar as seguintes informações: Título do Anúncio, Descrição completa, Cargo, Faixa Salarial, Cidade, Estado, Data, Código da Vaga e Nome da Empresa Confidencial

	4.1. Caso a vaga seja do tipo PREMIUM, o nome da empresa será apresentado de acordo com a configuração da empresa anunciante
	
	4.2. Caso o usuário seja anônimo, o site irá apresentar um botão "Cadastre-se ou faça o login para concorrer a esta vaga" caso a vaga seja FREE, ou um botão ""Vaga disponível apenas para usuário PREMIUM. Cadastre-se ou faça o login para concorrer a esta vaga".

		4.2.1 Caso o usuário clique no botão "Cadastre-se ou faça o login para concorrer a esta vaga", acionar "CDU010 - Autenticar"
		4.2.2 Caso o usuário clique no botão "Vaga disponível apenas para usuário PREMIUM. Cadastre-se ou faça o login para concorrer a esta vaga", acionar "CDU010 - Autenticar"

	4.3. Caso o usuário esteja autenticado como FREE, o site irá apresentar o botão "Concorrer a esta vaga" caso a vaga seja FREE ou o botão "Seja um usuário PREMIUM e concorra a esta vaga".

		4.3.1 Caso o usuário clique no botão "Concorrer a esta vaga", acionar "CDU004 - Concorrer Vaga"
		4.3.2 Caso o usuário clique no botão "Seja um usuário PREMIUM e concorra a esta vaga", acionar "CDU020 - Cadastrar Usuário"		

	4.4. Caso o usuário esteja autenticado como PREMIUM, o site irá apresentar o botão "Concorrer a esta vaga".

		4.4.1 Caso o usuário clique no botão "Concorrer a esta vaga", acionar "CDU004 - Concorrer Vaga"

5. O site terá um botão para retornar a tela anterior, mantendo o resultado da pesquisa informada anteriormente.

	5.1 Caso o usuário clique no botão de retorno à pesquisa, o site irá retornar à página de resultado da pesquisa, com cursor exatamente na posição original.



Requisitos
=========================================================================

O resultado da pesquisa deverá ser paginado e ordenado pelos seguintes critérios de pesquisa
	Tipo da Vaga (PREMIUM / FREE)
	Data da vaga
	Titulo
	Nome da Cidade
	Nome do Estado

O usuário poderá refinar a pesquisa realizada através dos critérios disponíveis no "CDU003 - Refinar Pesquisa"

O usuário poderá concorrer a vaga pesquisada de acordo com as regras disponíveis no "CDU004 - Concorrer Vaga"

As vagas do tipo PREMIUM deverão aparecer sempre nas primeiras posições e serem destacadas com borda e cor de fundo diferenciadas.

As vagas que estiverem inativas não serão apresentadas na pesquisa.

Os candidatos que estejam bloqueados por um empresa PREMIUM, não visualizarão vagas desta empresa no resultado de pesquisa.

As vagas que façam parte de empresas bloqueadas por um usuário PREMIUM, não serão apresentadas a este usuário.
